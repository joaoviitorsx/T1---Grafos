
# Projeto 2 - Grafos

## Descrição

Este projeto implementa conceitos fundamentais de teoria dos grafos, incluindo representações, algoritmos de busca e análise de estruturas de rede.

## Objetivos

- Implementar estruturas de dados para representação de grafos
- Desenvolver algoritmos clássicos de traversal (BFS, DFS)
- Analisar propriedades e características de grafos
